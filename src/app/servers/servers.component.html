string interpolation - between double curly bracket must be a string, a string variable or variable that can be converted to string, a method call that return a string, and a ternary operation. used to print a text

property binding - the attribute disabled of button is now dynamic by placing in square brackets and will change depend on the variable in quotes. used to change property of html element

event binding - event in parenthesis, ex: (click) and set value to an executing code

$event keyword is reserved for event data

two way data binding - add keyword [(ngModel)] and set it to a variable

structural directive that creates new element so *[directive] - *ngIf. attribute directive that updates element so [directive] - [ngStyle] and [ngClass]

[ngStyle] is set to an object whose key is css property (backgroundColor) and value is value for this prop (can use conditional statement to change style), [ngClass] is set to an object whose key is class name and value is a condition to apply this class
